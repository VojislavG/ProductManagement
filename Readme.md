
Derzeitige Möglichkeiten :
  -Man kann ein Produkt in DatenBank hinzufügen, bearbeiten oder löschen...
  -Jedes neue Produkt hat seinen eigenen Barcode
  -Die Produkte werden in eine Products-Box gelegt
  -Jede Box hat einen eigenen Barcode, der sich vom Produkt-Barcode unterscheidet
  -Jede Box hat maximale Größe von 10 Produkte...die werden automatisch erstellt
